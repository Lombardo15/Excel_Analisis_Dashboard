# Excel_Analisis_Dashboard
Objetivo
Analizar el comportamiento de transacciones de un ecommerce multicanal (Amazon, tienda propia, etc.) para identificar oportunidades de mejora en rentabilidad, retención de clientes y experiencia de compra.
Hallazgos Clave

Rentabilidad por canal: Se identificaron los canales y categorías de productos con mayor venta neta, permitiendo priorizar esfuerzos comerciales.
Retención de clientes: El análisis de cohortes reveló los meses críticos donde se pierde mayor volumen de clientes desde su primera compra.
Comportamiento de compra: El basket analysis mostró patrones de co-compra por categoría útiles para estrategias de cross-selling.
Satisfacción y devoluciones: Existe correlación entre el método de envío y el nivel de satisfacción del cliente; ciertos métodos presentan mayor tasa de devolución.
Patrones temporales: Se detectaron días y franjas horarias con mayor concentración de pedidos, lo que permite optimizar la operación logística.
Geografía: Se identificaron los mercados (países y regiones) con mejor performance en ventas netas.
Conversión y pagos: Se analizaron las combinaciones de método de pago y tipo de entrega más frecuentes en órdenes completadas.

Herramientas utilizadas
Excel (Tablas Dinámicas, Dashboard, Gráficos)
